# expense_tracker
A simple and efficient expense tracking application built using Python and SQLite. This project helps users manage their finances by recording, categorizing, and summarizing daily expenses.

## 🛠️ Features
- Add Expenses: Record your expenses with details like amount, name, and category 
- Track Income: Log your income sources for a clear view of your earnings.
- Categorization: Group expenses into categories for better analysis.
- Database Storage: Store all data securely in an SQLite database.
- Data Persistence: Ensures your data remains available even after restarting the app.

## 🧰 Technologies Used
- Python: Core programming language used for the logic and functionality.
- SQLite: Lightweight database for storing and managing financial records.

## 📈 Future Improvements
- Add a graphical user interface (GUI) for a better user experience.
- Enable exporting data to CSV or Excel files for detailed financial analysis.
- Include budget planning and expense forecasting features.
- Implement user authentication for multi-user support.
